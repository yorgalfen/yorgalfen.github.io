Help Page

This website is a visualization of the Liebnitz Beta Plateau, near the lunar South Pole, for the 2023 NASA App Development Challenge. For questions, email me using this link. View the source files at https://github.com/yorgalfen/yorgalfen.github.io.

Controls:

Click and drag to look around.

W Move forward
A Move left
S Move backward
D Move right
L Toggle automatic height adjustment. Starts off.
Q Move up, if automatic height adjustment is disabled.
E Move down, if automatic height adjustment is disabled.
C View position and other information.
P Input a position for teleportation. See below.
X Change rendering size. This number is the width, in data points, of the square rendered at one time. Default is 64.
M Change the slope, in degrees, above which squares will be marked in red. Default is 15.
N Change the slope, in degrees, below which squares will be marked in green. Default is 5.
H Display this help page.

Color Meanings:

Red - slopes greater than your selected value (15° by default)

Green - slopes less than your selected value (5° by default)

Yellow - route from landing to destination site

Blue - communication checkpoints along route

Teleportation:

Input a data file row and column (space-separated) to go to, or input latitude and longitude, space-separated. Note that coordinate-based teleportation may be inaccurate.

-85.7882225 < latitude < -85.0634174

27.289349 < longitude < 36.3651141
